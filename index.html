Conway's Game of Life

What is it?

Conway's Game of Life is a cellular automaton created by mathematician John Conway in 1970. It's a zero-player game where the evolution is determined by its initial state.

Rules

  • Birth: A dead cell with exactly 3 living neighbors becomes alive
  • Survival: A living cell with 2 or 3 neighbors stays alive
  • Death: A living cell with fewer than 2 neighbors dies (underpopulation)
  • Death: A living cell with more than 3 neighbors dies (overpopulation)

How to Use

  • Click on cells to toggle them alive or dead
  • Preset Patterns - Click a pattern button, then click the grid to place it
  • Start/Pause button to control the simulation
  • Speed slider to adjust how fast generations evolve (1-10 per second)
  • Clear Grid to remove all living cells
  • Randomize to create a new random pattern (30% filled)
  • Add 100 Cells to add 100 random living cells to the current grid

Pattern Types

  • Spaceships: Glider, LWSS - Move across the grid
  • Oscillators: Blinker, Toad, Beacon, Pulsar, Pentadecathlon - Repeat patterns
  • Guns: Glider Gun - Creates a stream of gliders
